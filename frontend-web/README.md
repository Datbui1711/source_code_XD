# CareerMate Web Application

React-based web application for CareerMate platform.

## Features

- ✅ User Authentication (Login/Register)
- ✅ Dashboard with stats
- ✅ Responsive design
- ✅ API integration with backend

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Development

The app will run on http://localhost:3000

API calls are proxied to http://localhost:9090 (API Gateway)

## Pages

- `/login` - Login page
- `/register` - Registration page
- `/dashboard` - Main dashboard (protected)

## Tech Stack

- React 18
- Vite
- React Router
- Axios
- CSS3
